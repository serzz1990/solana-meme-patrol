Solana MEME Patrol

Your ultimate plugin for tracking new tokens on specific Solana wallets and receiving Telegram notifications!

What is Solana MEME Patrol?

Solana MEME Patrol is a simple yet powerful Chrome extension designed to help you monitor Solana wallets. It sends real-time notifications to your Telegram when new tokens are added to any of the wallets you track.

How to Use It?

  1. Get a Telegram Bot Token:

    Create a bot on Telegram via BotFather. Copy the API token.

  2. Find Your Telegram Chat ID:

    Use your bot to send a message to yourself. You can find your Chat ID via getUpdates.

  3. Install and Configure the Plugin:

    Enter your Telegram Bot Token, Chat ID, and the Solana wallet addresses you want to track.

  4. Save the Configuration:

    Click the Save button. The plugin will start monitoring and send Telegram notifications whenever new tokens are detected.

Why Use Solana MEME Patrol?

  • Instant notifications for new tokens.
  • Track multiple Solana wallets simultaneously.
  • Simple setup and user-friendly interface.
